Enfoque del tarot sí o no

El tarot sí o no se utiliza cuando una persona necesita una orientación rápida sobre una pregunta concreta. La respuesta puede ser favorable, negativa, temporal, condicionada o prudente según la carta y el contexto.

Este repositorio no busca sustituir una lectura extensa, sino organizar un método básico para leer el resultado con más claridad.
